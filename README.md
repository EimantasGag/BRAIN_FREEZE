# MEMORY GYM by BrainFreeze


## Functionalities
1. Alfa version
    - Have one finished game
2. Beta version
    - Add more games to the gym. (reach at least 3)
3. Final version
    - Login screen
    - Saving the points aquired by playing to the accounts 
    - Leaderboard
